# Sistema de Gestión - Tienda de Música CEA
## Incremento del Sprint 1 - Producto Mínimo Viable

Este proyecto contiene la simulación de los módulos principales para la gestión operativa de una tienda de discos musicales.

### Funcionalidades implementadas:
- **HU-005:** Registro y gestión de perfiles para artistas y agrupaciones.
- **HU-003:** Catálogo maestro para el alta de obras musicales con códigos UPC.
- **HU-002:** Terminal de punto de venta con lógica de descuento de inventario.

### Tecnologías utilizadas:
- **Backend:** Spring Boot (Estructura de controladores y modelos).
- **Frontend:** HTML5 y Tailwind CSS para la interfaz interactiva.
- **Gestión Ágil:** Metodología Scrum documentada en Trello.

### Instrucciones de uso:
Para visualizar el prototipo funcional basta con abrir el archivo `src/main/resources/static/index.html` en cualquier navegador moderno. La interfaz permite simular el flujo completo de los casos de prueba definidos en el documento final.
